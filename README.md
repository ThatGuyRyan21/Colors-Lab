# Colors-Lab
The Colors Application is used to add colors in a user personal account that can be searched and returned to the user at any time. 

# Technologies Used
  Github/Git
  Powershell
  VS Code
  PHP
  MYSQL

# Setup
  1. Navigate to Powershell/Terminal
  2. Type "git clone https://github.com/"GITHUB USERNAME"/Colors-Lab.git"
  3. Navigate to "cd Colors-Lab"
  4. Type git remote -v
Now you have access to the repo
